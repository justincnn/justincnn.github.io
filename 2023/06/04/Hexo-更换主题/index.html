Hexo除了可以自由部署外，吸引人的另外一点就是多样的个性化主题，有谁可以在21世纪对个性化产生抗性呢，接下来给大家演示下如何更换主题：

后面的内容在首页不显示，只显示到这里

安装主题

到主题的 GitHub 主页下，复制地址

进入博客根目录：输入命令： git clone 刚才复制的地址 themes/xxxx ，xxxx 是主题名
例如：

或者：从 https://hexo.io/themes/ 下载 release 包，解压到博客根目录 themes/ 文件夹下

使用主题

修改博客根目录下的 _config.yml 文件，注释掉原来的 theme 并新增一句 theme: xxxx （注意冒号后面有空格）

xxxx 为博客根目录 themes/ 文件夹下，对应的主题文件夹名称。
