打造云openclaw适合你的钢铁侠专属贾维斯

看钢铁侠的时候，总觉得“有个贾维斯”只是幻想：一个只为你服务、了解你习惯、懂你脾气的私人助手。现在，用 openclaw，我们可以开始搭建这样一个“个人专属贾维斯”——它会在和你一起工作的过程中，自动学习你关心的知识、适应你的表达方式和决策偏好，越用越像你，既能给你启发当老师，又能陪你讨论当朋友，甚至像徒弟一样替你跑腿、干活。接下来这篇教程，我会带你从零开始，一步步把这个会成长、会进化的专属助手搭出来。

简介

如果把你的电脑、服务器、各种软件，想象成一堆“会干活的工具”，那 OpenClaw 更像是坐在中间那个“大总管”——它帮你点鼠标、点按钮、跑流程，但你自己不用再一遍遍去点。

更特别的是，OpenClaw 给应用和系统加了一层“现代化遥控器”。它不要求你守着本地电脑，人必须在场，而是可以运行在远程机器上（比如 VPS、云服务器、公司机房里的电脑），能随时随地远程查看和控制这些自动化流程。
这意味着：想要稳定的自动化、统一的远程管理，不用再担心“家里电脑不能关机”这种问题——OpenClaw 会一直在后台替你守着、替你干活。

OpenClaw 的用例

比如这些常见场景，都可以交给你的“个人贾维斯”来干活：

  • 做调研报告：你只用给主题、问题和截止时间，它自动帮你搜资料、筛选重点、列出结构化大纲，再生成一版可改的初稿。
  • 写周报 / 汇报：根据你的日程、聊天记录、文档变更，自动提取本周成果，按模板排版成周报或汇报提纲。
  • 客户跟进：统一记录不同平台的沟通内容，自动生成跟进清单、提醒时间，甚至帮你草拟下一条回复。
  • 生活场景：做旅行计划、比价买电子产品、整理孩子学习资料，它都能先帮你“打底”，你只做最后决策。

  • 如果你打算长期跑多个频道、开浏览器自动化、让助手一直在线，更建议直接按推荐配置来准备机器，稳定性和流畅度会好很多。

安装技能

技能为 OpenClaw 增添了额外的功能和集成。OpenClaw 自带 50 多种技能。您可以在仪表板中直接安装技能，方法是打开“技能”部分并选择要启用的技能。

启用钩子

钩子允许 OpenClaw 在特定事件发生时或在使用特定命令时自动执行操作。这对于自动化重复任务、保存上下文或触发无需人工干预的工作流程非常有用。

安全问题

OpenClaw 功能很强，但也意味着一定要重视安全。

  • 网关仪表板是用一个「令牌（token）」来保护的，它就相当于你的登录密码，谁拿到它，谁就能直接控制你的 OpenClaw。
    务必不要在截图、工单、聊天记录等任何公开场合泄露这个令牌。

  • 最推荐的做法，是让 OpenClaw 待在防火墙后面，不要把网关界面直接暴露在公网。
    默认情况下，OpenClaw 会把网关服务绑定到 127.0.0.1，也就是只能在服务器本机访问——这对绝大多数人来说，是最安全的设置。

  • 如果你需要在自己电脑上远程访问控制面板，做法是：
    使用 SSH 端口转发 建立一个加密通道，只把访问权限给到你信任的少数用户，而不是对整个互联网开放。

获取telegram bot id和siliconflow api

telegram

在 Telegram 里使用 OpenClaw，第一步是先创建一个 Telegram 机器人，并拿到它的 API Token。这个过程不需要写代码，直接在 Telegram 里和官方账号 BotFather 对话就能完成。

先打开 BotFather，发送 /newbot，然后按提示操作：

  1. 输入机器人显示名称（Name）
  2. 输入机器人用户名（Username）
    • 注意：用户名必须以 bot 结尾，例如 my_assistant_bot

创建成功后，BotFather 会返回一段类似下面的信息：

  • 你的机器人链接：t.me/你的机器人用户名
  • 你的 API Token：xxxxxxxxxxxxxxxxxxxxxxxx

这个 Token 就是 OpenClaw 连接 Telegram 的“钥匙”，后面要填到配置文件里的 botToken 字段。
务必保管好，不要发到群里、工单或截图里。任何拿到 Token 的人，都可能控制你的机器人。

siliconflow API

你可以通过这个链接注册 SiliconFlow：
https://cloud.siliconflow.cn/i/O4301wgk
也可以直接扫码注册。

注册登录后，进入控制台按图示创建并获取 API Key。拿到后复制到 OpenClaw 配置里的 apiKey 字段即可。

提醒：API Key 不要发到群聊、截图或公开文档里。
